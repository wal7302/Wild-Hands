# Wild Hands

## Milestone 200

Today Wild Hands stopped being a console application.

It became a game engine.

From this point forward the engine produces scenes.

Renderers decide how players experience those scenes.

This architecture allows the same game to run as:

- Console
- Windows
- Mac
- iPhone
- Android
- Web
- Future platforms

without changing the game rules.

This is the point where Wild Hands became platform independent.
